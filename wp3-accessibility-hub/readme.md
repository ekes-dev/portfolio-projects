# Werkplaats 3: Accessibility Hub

## Beschrijving

Een platform waar ervaringsdeskundigen zich kunnen inschrijven voor toegankelijkheidsonderzoeken. Inclusief beheerdersportaal en REST API.

## Tech stack

- Python
- Flask
- REST API
- HTML/CSS/JS
- bcrypt

## Mijn bijdrage

- Meewerken aan backend flow rondom registratie + login
- Opzetten en testen van REST-routes
- Inrichten van dashboardfunctionaliteit
- Oplossen van problemen met authenticatie, validatie en API calls
- General code cleanup en structuurverbetering

## Wat ik heb geleerd

- REST API's ontwerpen en testen
- Form validatie + dynamische front-end componenten
- Toegankelijkheidseisen (WCAG) toepassen
- Samenwerken aan een grotere Python codebase
