# Werkplaats 2: Vragen Indexeer Applicatie (VIA)

## Beschrijving

Een Flask-webapplicatie waarmee redacteuren toetsvragen kunnen indexeren op basis van taxonomie. Met AI-analyse wordt een voorstel gegenereerd voor het moeilijkheidsniveau.

## Tech stack

- Python
- Flask
- bcrypt
- OpenAI API
- HTML/CSS
- Jinja
- SQLite

## Mijn bijdrage

- Implementeren van meerdere features in de vraag-indexeerflow
- Meebouwen aan AI-promptlogica voor het automatisch bepalen van taxonomieniveau
- Meewerken aan login- en rollenstructuur
- Bugfixing en structuur verbeteren waardoor de applicatie stabieler en duidelijker werd

## Wat ik heb geleerd

- Werken met Flask (routing, templating, authenticatie)
- Integratie met LLM-API's
- Teamwork in een MVC-achtige Python setup
