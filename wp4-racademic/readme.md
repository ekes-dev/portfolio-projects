# Werkplaats 4: RACademic

## Beschrijving

Een platform waar HR-studenten studiematerialen kunnen delen. Met FastAPI backend (JWT-auth), React Native frontend en Docker-omgeving.

## Tech stack

- FastAPI
- React Native (Expo)
- SQLite
- Axios
- Docker

## Mijn bijdrage

- Backend routes implementeren (bron-beheer, stemmen, authenticator)
- JWT-auth opzetten voor zowel web als mobiel
- Maken van frontend functionaliteiten zoals favorieten, bronoverzicht & filters
- Oplossen van cross-platform problemen (web/iOS/Android)
- Meedenken over architectuur en projectstructuur

## Wat ik heb geleerd

- Full-stack development met FastAPI + React Native
- Token-based authentication
- Docker workflow
- Complexere front-end state management (useContext, navigatie, etc.)
