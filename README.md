# Portfolio Projects

Deze repository bevat projectbeschrijvingen van verschillende schoolprojecten die ik tijdens mijn AD Software Development opleiding heb gebouwd.  
De focus ligt op mijn bijdragen, gebruikte technologieën, screenshots en wat ik per project heb geleerd.  
Dit is bewust geen code-repo, maar een overzicht van mijn werk.


## Projecten

### 1. Vragen Indexeer Applicatie (Werkplaats 2)
AI-gestuurde applicatie voor het indexeren van toetsvragen op basis van taxonomie.  
**Technologieën:** Python, Flask, OpenAI API  
**Mijn rol:** vraaganalyse, indexeerflow, login/rollen, bugfixing  
**Map:** `/wp2-vragen-indexeren`


### 2. Accessibility Hub (Werkplaats 3)
Platform voor ervaringsdeskundigen en organisaties om toegankelijkheidsonderzoeken te beheren.  
**Technologieën:** Python, Flask, REST API, JavaScript  
**Mijn rol:** backend routes, dashboardfunctionaliteit, validatie, API testing  
**Map:** `/wp3-accessibility-hub`


### 3. RACademic (Werkplaats 4)
Full-stack platform waar studenten studiemateriaal kunnen delen. FastAPI backend met JWT-authenticatie en een React Native frontend.  
**Technologieën:** FastAPI, React Native (Expo), Docker, Axios  
**Mijn rol:** backend functionaliteit, JWT-auth, frontend features, cross-platform fixes  
**Map:** `/wp4-racademic`
